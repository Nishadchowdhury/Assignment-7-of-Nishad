js সাধারণত ২ ভাবে run করে node.js এবং Browsers এর মধ্যে , তবে উভয় ক্ষেত্রেই js Engine সাহায্যে কাজ করে । Engine এর কাজ হল human readable js formated sheet এর মধ্যে থাকা code কে machine readable কোড এ convert করে (01) । Engine এর মধ্যে 2 টা part থাকে 1. Memory Heap (To Store data for Execution) 2. Call stake (Where all functions are stored as a stake) এটা এটা হল LOFO (last in fast Out) এটার মধ্যে data উপর উপর করে রাখা হয় (which call stake frame) , যে data শেষে আসে সেটা আগে Execute হয় । call stake এর মধ্যে যদি কোন Asynchronous data চলে আসে তবে সেটাকে WEB api এর কাছে রেখে পরের Synchronous data একের পর এক Execute করতে থাকে । এই জন্য JS কে বলে single-threaded programming language অর্থাৎ এটা একক সময়ে একটার বেশি কাজ করতে পারেনা । কিন্তু এটা যেহেতু Asynchronous data কে web api এর কাছে দিয়ে সে তার কাজ করতে পারে তাই runTime এর মধ্যে একক সময়ে একাধিক কাজ হতে থাকে , তাই এটাকে বলে event driven programming language । js একটা Client side programming language বলার কারন হল এটা file আকারে Client এর conputer এর মধ্যে download হয় এবং Client এর conputer এর মধ্যেই run হয় ।
